🧩 是什么
hucre 是一个零依赖的电子表格引擎，用纯 TypeScript 编写，可运行于任何 JS 环境（浏览器、Node.js、Edge Runtime、Web Worker 等）。
💡 "hucre" 是土耳其语中 "单元格" 的意思，即 Excel 里的 cell。
⚡ 快速安装
$ npm install hucre
💻 示例代码
TypeScript
import { readXlsx, writeXlsx } from "hucre";

// 读取 XLSX 文件
const workbook = await readXlsx(buffer);
console.log(workbook.sheets[0].rows);
